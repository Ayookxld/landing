# Through Steam

Open Steam Make sure you have the games installed you want to play. Right click onto the game

  • CoD: MW3 - Multiplayer
  • CoD: BO2 - Multiplayer
  • CoD: BO2 - Zombies
  • CoD: WaW

Select "Manage" then "Browse local files" A new explorer window opens with the game files.

# Through Plutonium

Open the launcher
Select the game you are looking for.
Click on the cog wheel next to the "Play" button.
Select the path and copy it.
Press the Windows key + r, paste the path and then press OK / return.
